# Sistema de Gerenciamento de Tarefas

Sistema desenvolvido para a disciplina de Programação Orientada a Objetos I. Implementa os conceitos fundamentais de POO com Java, incluindo classes, encapsulamento e herança.

## Funcionalidades
- Cadastro de tarefas
- Consulta por status
- Atualização e exclusão
- Validações de dados

## Tecnologias
- Java
- Paradigma Orientado a Objetos

*Projeto acadêmico do primeiro bimestre de POO I*
